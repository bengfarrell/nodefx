Browser based Microphone FX manager

An easy way to try effects with microphone input is to run entirely in your browser using the Web Audio API.

This fakes a web socket connection to a server, so it shares the same code as the node version, but runs in the browser.

Try it out

Node based Microphone FX manager

The same UI as above, but establishes a real web socket connection to a Node and node will create the microphone input and apply the effects all in Node using the "node-web-audio-api".
